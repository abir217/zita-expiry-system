# zita-portal-final (React + Vite)

This is a simple, static-styled React + Vite project for the ZITA Telecom Recharge Expiry page.
It's configured for GitHub Pages under the path `/zita-portal-final/`.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview the production build:
```bash
npm run preview
```

## Deploy to GitHub

1. Create a repository named `zita-portal-final` on GitHub.
2. Push this project to `main` branch.
3. In GitHub repo settings → Pages:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/dist`

Your site will be available at:
`https://<your-username>.github.io/zita-portal-final/`
